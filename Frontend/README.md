# Event Discovery Platform Frontend

React + Vite JavaScript frontend for an event discovery and notification platform.

## Start

```bash
cd event-discovery-platform/frontend
npm install
npm run dev
```

## Stack

- React JavaScript
- React Router
- Axios
- TanStack Query
- React Hook Form
- Zod
- Tailwind CSS
- Lucide React icons

## Main folders

```text
src/app          App providers, router, layouts
src/shared       Reusable components, API client, constants, utilities
src/features     Feature modules and pages
```
